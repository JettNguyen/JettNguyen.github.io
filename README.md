# [JettNguyen.github.io](https://jettnguyen.github.io/)

Personal portfolio site for my projects, coursework, reports, presentations, and resume content.

## Features

- Browse project case studies with problem, approach, and outcome structure.
- Look over my University of Florida public coursework portfolio and report archives.
- Watch presentations with slides and/or videos.
- View my digital resume and/or download resume file.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Static export (`next build` → `/out`)
